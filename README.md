# Netflix Data Analysis using Python & Matplotlib

## Project Overview
This project analyzes Netflix content using Python, Pandas, and Matplotlib.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Analysis Performed
1. Data Cleaning
2. Movies vs TV Shows Distribution
3. Netflix Content Growth Over Years
4. Top 10 Countries Producing Content
5. Top Genres on Netflix
6. Content Ratings Analysis
7. Content Distribution Pie Chart

## Key Insights
- Movies dominate Netflix content.
- Content production increased significantly after 2015.
- United States contributes the highest number of titles.
- Drama and International Movies are among the most common genres.

## Dataset
Netflix Titles Dataset (Kaggle)
